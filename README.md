# 🚀 HADIR Pro --- Attendance Data Cleaner

HADIR Pro adalah aplikasi web sederhana untuk **membersihkan,
menganalisis, dan memvisualisasikan data absensi karyawan** dari file
CSV maupun Excel (XLS/XLSX).

------------------------------------------------------------------------

## ✨ Fitur Utama

-   📂 Import file CSV / XLS / XLSX
-   🧹 Pembersihan data otomatis
-   ⏱️ Perhitungan durasi kerja & keterlambatan
-   📊 Dashboard analitik
-   📥 Export ke CSV

------------------------------------------------------------------------

## 🛠️ Teknologi

-   HTML, CSS, JavaScript
-   SheetJS (xlsx)
-   Chart.js
